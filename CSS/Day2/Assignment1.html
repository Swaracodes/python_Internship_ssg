Note:

Difference between inline and block elements:

  • Inline: Does not start on a new line and only takes as much width as necessary. Cannot set width/height.
  • Block: Always starts on a new line and takes full width. You can set width/height.
  • Inline-block: Behaves like inline (sits in line), but allows width/height like block elements.
